# 🎨 VLESS XHTTP Config Generator

> **A simple web tool to generate optimized VLESS-XHTTP links for Netlify Relay.**

---

## 📌 About the Main Project

This website is a companion tool for the **Netlify XHTTP Relay** project. It is designed to make generating VLESS configuration links easier and faster without manual editing.

For the backend relay functionality (Edge Functions), please refer to the main repository:

👉 **[Netlify XHTTP Relay - Main Project](https://github.com/amirs/Netlify-XHTTP-Relay)**
---

## 🌐 About This Tool

This web application allows you to:
- Generate valid **VLESS** links with **XHTTP** transport.
- Customize UUID, Netlify Domain, Backend Path, SNI, and Address.
- Copy the final link instantly with one click.

It is the perfect companion for users of the Netlify XHTTP Relay project who want to create configs quickly.
